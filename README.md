# web-app

Web app rust tauri example.

* [tauri](https://tauri.app/)
* [vue](https://vuejs.org/)
* [vuetify](https://vuetifyjs.com/en/)


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
